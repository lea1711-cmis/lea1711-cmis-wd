Ordered List

  1. Create a new file in your repository.
  2. Type in gedit filename to open up gedit and edit your work.
  3. Type in correct html code.
  4. Validate on validator.w3.org
  5. Check to see if the file appears on localhost:8000/filename.
  6. Save your work in gedit.
  7. Type in "python -m SimpleHTTPServer" onto terminal so your work can be viewed, then check to see if the webpage can be viewed by typing in, localhost:8000/filename.
  8. Type in sftp (type in sftp username@cmis-hsteacher, then your password) onto terminal.
  9. Then, type in cd public_html and then "put filename"
  10. In terminal, type "git add filename," then "git commit -m "commit message"," then "git push." This adds and saves the webpage onto github.com
